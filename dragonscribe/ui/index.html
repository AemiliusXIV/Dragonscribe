The game is running. Close it completely before editing; the game caches saves in memory and will overwrite your changes (or corrupt the file). Editing is disabled until it's closed.

Worlds

Converting a world to Custom unlocks all difficulty sliders, but disables achievements while it stays custom. Reverting to Standard restores them. Custom worlds need characters set to Custom below; the two must match. A heads-up from the original tool: conversion can disturb locked chests, protection totems and privacy settings, so clear those first.

Characters

A character's compatibility must match the world it joins: Standard for Standard/Hard worlds, Custom for Custom worlds. The setting is mutually exclusive; a Custom character can't join Standard worlds, and vice versa.

Advanced: Character Skills
